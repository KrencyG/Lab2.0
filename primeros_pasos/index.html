Primeros Pasos

Crear un Hola Mundo con Angular es muy sencillo. De hecho Angular al crear un proyecto ya te crea una especie de Hola Mundo. Lo primero va a ser abrir VSCode y situarnos en la carpeta donde queramos crear el nuevo proyecto, una vez hecho eso usaremos el siguiente comando: ng new <nombre-del-proyecto> Por ejemplo: ng new HolaMundo

Una vez le demos a enter, se nos daran una serie de opciones de configuracion, por ejemplo si queremos usar CSS o si queremos utilizar algun bot de IA. Aqui simplimente lo configuraremos segun lo que necesitemos y nuestras preferencias personales.

Una vez que terminemos con la configuracion que Angular nos pide veremos que se nos ha creado una carpeta llena de archivos, no hay que asustarse por ver tantos archivos y carpetas. Antes de pasar a entender que son cada archivo y cual es su funcion vamos a comprobar que todo funciona, para ello vamos a probar en local la web que acabamos de crear. Para ello usaremos el siguiente comando: ng serve Es importante que nos aseguremos que estamos en la carpeta que acabamos de crear, si no nos dara un error.
