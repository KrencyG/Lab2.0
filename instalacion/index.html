¿Como puedo Instalar Angular?

¿Se precisa instalacion? ▶

Si es necesario descargar tanto Angular como algunos otros componentes necesarios para que funcione como Node.js y npm

¿Que pasos hay que realizar para instalar Angular? ▶

Instalar Angular es muy sencillo, lo primero que necesitaremos es tener un editor de texto estilo Visual Studio Code y tener Node.js instalado, eso se puede hacer desde este enlace: https://nodejs.org/es/download/ Una vez que ya hecho eso tendremos que abrir VSCode y en el terminal instalar Angular con el siguiente comando: npm install -g @angular/cli

Versiones necesarias ▶

En cuanto a las versiones que vamos a usar se recomienda usar la ultima version de Angular y por lo menos la version 20.19 o superior de Node.js

Variables de entorno ▶

Debemos tener en cuenta que para usar los comandos de Node.js es necesario que npm este dentro de las variables de entorno. Para ello iremos a Path e incluiremos npm. Podemos ver si funciona abriendo un terminal y usando el comando node -v

Base de datos ▶

No es necesaria ninguna base de datos ni ninguna otra libreria externa, solo con lo anterior es suficiente
