Utilización de Angular

Una vez que ya tenemos nuestro proyecyo creado y vemos que todo funciona podemos pasar al siguiente paso. Vamos a explorar un poco todos los archivos que ha creado Angular, la mayoria de ellos no son 'necesarios' como tal son simplemente decoraciones para que la pagina del hola mundo se vea asi. Por ejemplo vamos a buscar el archivo app.html, dentro de el es donde esta el codigo de todo lo que vemos en el hola mundo, lo podemos borrar si queremos y poner una etiqueta con un Hola Mundo. Si guardamos este cambio automaticamente se nos habra modificado en la pestaña que teniamos abierta sin necesidad de volver a utilizar ng serve, de este modo podemos ver de forma muy rapida y directa todos los cambios que vayamos realizando.

A partir de aqui ya podemos hacer lo que queramos segun la web que queramos desarrollar, por ejemplo cosas recomendables pueden ser el crear un header y un footer. Para mantener un poco el orden vamos a utilizar el siguiente comando: ng generate component <nombre carpeta> Yo por ejemplo en ejemplo de la foto he creado unas carpetas con mkdir y luego he creado los compoenentes header y footer con el comando ng generate component

Esto como tal no hace nada, si vemos la pagina web, veremos que no hay ningun cambio nuevo, para que se incluya el header y el footer tendremos que inccluirlos tanto en el HTML con las etiquetas que ha creado Angular como en el TypeScript. Esta ultima parte puede ser algo mas compleja, pero simplemente le tenemos que decir al TS que importe tanto el header como el footer desde donde se encuentren, por ejemplo asi: import { Header } from './components/header/header' Tambien tenemos que incluirlo en el array de imports que tenemos en el TS.
