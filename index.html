¿Que es angular y como funciona?

¿Para que sirve? ▶

Angular es un framework de Google que esta enfocado en crear aplicaciones de una sola pagina (SPA). Su lenguaje principal es TypeScript. Ademas ofrece otras herramientas como: •Enrutamiento •Inyeccion de dependencias •Formularios reactivos •Sistema modular Angular se diferencia de otros frameworks en que es un framework completo, no es una biblioteca como otros. Esto permite que angular ofrezca funcionalidades como routing, HTTP, validaciones... sin el uso de librerias externas.

¿Que aplicaciones tiene? ▶

El uso principal de Angular es la creacion de SPA, es decir, paginas que cargan una unica vez al principio y una vez hecho esto no tienen que recargar nada para navegar entre secciones. Esto hace que sea mucho mas rapido y fluido

¿Sobre que lenguajes se apoya? ▶

Angular utiliza 3 lenguajes principales: •HTML: Para definir las plantillas y los componentes •CSS: Para el estilo y la presentacion de los elementos •TypeScript: Para la logica y funcionalidad de los componentes

¿Que prerrequisitos necesita? ▶

Como para diseñar la mayoria de paginas webs, es necesario tener conocimientos de HTML, CSS Y JavaScript. La novedad que incluye Angular es el uso de TypeScript y Node.js y npm para la gestion de los paquetes.
